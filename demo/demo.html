Welcome to UTPlay.

YouTube Playlist with class - This simple script will allow you to integrate any YouTube Public Playlist into a CSS3 and Javascript audio player. YouTube Playlist will allow you to get only the sound out of your videos by hiding the block of the video frame. If your having issues playing your playlist, Please make sure your playlist is not private.

Submit a bug | Read the Official MD

Install

  • Download files from src folder.
  • Include files inside src on your page. <link rel="stylesheet" type="text/css" href="src/utplay.min.css"> <script src="src/utplay.min.js"></script>

How to use it

Simple, After adding the script dependencies use the html markup to add your player in place.

Now this is the only part that gets a bit tricky, You will need to find your YouTube Playlist ID and place it within the HTML markup of Iframe.

For my example, I am going to take a playlist I've created for this project https://www.youtube.com/playlist?list=PLr7KBbNMX5CTXyY1_GKjHAzrIPJRs6Cpo so my ID for this playlist will be PLr7KBbNMX5CTXyY1_GKjHAzrIPJRs6Cpo. Yes so pretty much everything after playlist?list=This_Is_Your_Id.
